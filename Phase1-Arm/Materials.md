## 🧰 Phase 1 — Materials Checklist

### Required

#### Actuation
- [ ] 3× NEMA 17 stepper motors  
- [ ] 3× stepper motor drivers (A4988 / DRV8825)  
- [ ] Driver heat sinks  

#### Control & Power
- [ ] Arduino Mega / Uno or ESP32  
- [ ] 12–24V DC power supply (5–10A)  
- [ ] USB cable  
- [ ] Breadboards  
- [ ] Jumper wires  

#### Mechanical
- [ ] 3D-printed or aluminum arm links  
- [ ] Base plate (rigid)  
- [ ] Bearings / bushings  
- [ ] Shaft couplers  
- [ ] Screws, nuts, washers  

#### Transmission
- [ ] GT2 belts + pulleys **or**  
- [ ] Spur gears  

#### Safety & Referencing
- [ ] 3× limit switches  
- [ ] Switch mounting hardware  

#### Tools
- [ ] Digital calipers  
- [ ] Multimeter  
- [ ] Hex keys / screwdrivers  
- [ ] Wire cutters  

#### Software
- [ ] Python + PySerial  
- [ ] Firmware IDE  
- [ ] GitHub repo  

---

### Optional (but Helpful)
- [ ] Cable management (zip ties, sleeves)  
- [ ] Enclosure for electronics  
- [ ] Oscilloscope (if available)
