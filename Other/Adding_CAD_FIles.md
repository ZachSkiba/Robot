# Adding CAD Files to a VS Code Folder

1. **Open VS Code** and open your project folder.

2. **Locate your CAD files** on your computer (e.g., `.step`, `.sldprt`, `.iges`).

3. **Copy or drag them** into your VS Code folder in the file explorer panel.

4. **Verify** the files appear in VS Code’s explorer.

5. **Optional:** Commit to GitHub:

```bash
git add .
git commit -m "Added CAD files"
git push
```
