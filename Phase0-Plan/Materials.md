# Phase 0 – Materials

## **Hardware**
Phase 0 is primarily simulation and planning, so minimal hardware is required. Optional components for reference or small tests:

- Laptop or desktop computer  
- Notebook + pen / sketching tools (for mechanical design & derivations)  
- Arduino board (Uno, Mega, or compatible) – optional for reference testing  
- Stepper motor + driver (A4988, DRV8825) – optional for verification  
- USB cable (for Arduino communication) – optional  
- Basic tools (screwdrivers, pliers, multimeter) – optional for reference hardware checks  

---

## **Software**
- Python 3.10+  
- Libraries:  
  - `numpy` – numerical computations  
  - `matplotlib` – plotting and visualization  
  - `scipy` – numerical IK / optimization  
- Jupyter Notebook or VS Code – interactive coding and plotting  
- Git – version control  
- GitHub repository – code and documentation backup  

---

## **Optional / Nice-to-Have**
- MATLAB + Simulink – cross-check kinematics or simulate arm dynamics  
- LaTeX or Markdown editor – professional documentation  
- Visualization tools (e.g., Blender, CAD software) – optional for arm sketches or plotting enhancements  

---

## **Consumables / Notes**
- No physical motors or electronics are strictly required in Phase 0; small reference testing is optional.  
- Focus is on **simulation, kinematics, trajectory planning, and logging**.  
- Documentation and sketches are critical for Phase 1 hardware implementation.
