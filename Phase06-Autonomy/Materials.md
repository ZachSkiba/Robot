# Phase 6 — Materials Checklist

## Required

- [ ] Fully working Phase-5 arm  
- [ ] Mobile base chassis  
- [ ] DC motors or steppers for base  
- [ ] Motor drivers  
- [ ] Wheel encoders  
- [ ] Power system (battery + regulation)  
- [ ] Onboard computer (ESP32 / SBC)  
- [ ] Python control stack

- [ ] IMU  
- [ ] Range sensing (ultrasonic / lidar-lite)  
- [ ] Rigid arm-to-base mount  
- [ ] Emergency stop  

## Optional (Advanced)

- [ ] SLAM  
- [ ] Multi-object tasks  
- [ ] Learning-based navigation
