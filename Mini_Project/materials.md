# Mini-Projects – Materials List (Phase 0)

## **General / Shared Materials**
| Item | Quantity | Notes |
|------|---------|------|
| Arduino Mega or ESP32 | 1 | Microcontroller for Mini-Projects A & B |
| USB cable | 1 | Connect microcontroller to computer |
| Breadboard | 1 | Temporary prototyping |
| Jumper wires | 20–30 | Signal connections and sensor wiring |
| Computer with Python | 1 | Data logging, plotting, controller interface |
| Python libraries | N/A | matplotlib, pyserial, numpy (for plotting/logging) |
| Power supply (5–12V, 2–5A) | 1 | Motor supply, stable voltage required |
| Basic tools | As needed | Screwdriver, pliers, wire strippers, etc. |
| Multimeter | 1 | Debugging voltages/currents |

---

## **Mini-Project A – Single-Axis Smart Joint**
| Item | Quantity | Notes |
|------|---------|------|
| Servo motor or stepper motor | 1 | High torque enough for joint test |
| Motor driver (if using stepper) | 1 | L298N, TB6600, or similar |
| Encoder or step-count sensor | 1 | For feedback loop |
| Screws, nuts, washers | Few | Mount motor securely |
| Small platform or bracket | 1 | Hold motor in place for testing |
| Heat shrink tubing / insulation | As needed | Protect wiring |
| Optional: 3D printed bracket | 1 | If motor mount not metal |

---

## **Mini-Project B – Controller → Motor**
| Item | Quantity | Notes |
|------|---------|------|
| PS5 controller | 1 | Input device for joystick control |
| USB dongle / Bluetooth adapter | 1 | Connect PS5 controller to computer or microcontroller |
| Same motor and driver as Mini-Project A | 1 | Reuse hardware |
| Wires, connectors | As needed | Connect controller input to logging/motor |

---

## **Mini-Project C (Optional) – 2-Link Python Simulation**
| Item | Quantity | Notes |
|------|---------|------|
| Computer with Python | 1 | Simulation environment |
| Python libraries | N/A | matplotlib, numpy (for FK/IK visualization) |
| Optional: Jupyter Notebook | 1 | Interactive simulation and plotting |

---

**Notes:**  
- These materials are **strictly scoped for mini-projects only** — Phase 1 full-arm hardware is **not included**.  
- Mini-Project A and B reuse much of the same motor, driver, and wiring.  
- Keep a few **spare wires and connectors** — early testing often requires adjustments.  
- Tools listed are **common hand tools**; no heavy machinery needed.  
- Optional 3D-printed mounts help but are not mandatory; simple platforms/brackets are sufficient.
